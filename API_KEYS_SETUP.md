# Настройка API ключей

API ключи для STT провайдеров читаются из переменных окружения (.env файл).

## Настройка

1. Скопируйте `.env.example` в `.env`:
   ```bash
   cp .env.example .env
   ```

2. Откройте `.env` и добавьте ваши API ключи:
   ```bash
   ASSEMBLYAI_API_KEY=ваш_api_ключ_assemblyai
   DEEPGRAM_API_KEY=ваш_api_ключ_deepgram
   ```

3. Файл `.env` автоматически игнорируется git (добавлен в `.gitignore`), так что ваши ключи в безопасности

## Получение API ключей

- **AssemblyAI**: https://www.assemblyai.com (бесплатный пробный период)
- **Deepgram**: https://www.deepgram.com (бесплатные кредиты при регистрации)

## Использование

После настройки просто выберите нужный провайдер в настройках приложения - API ключи будут загружены автоматически из переменных окружения.
